# مشروع عيادة الشفاء الرقمي

هذا المشروع عبارة عن نظام ويب متكامل لإدارة المواعيد الطبية، صُمم لتسهيل عملية حجز المواعيد للمرضى وتنظيمها للأطباء.

## الميزات الرئيسية
- **نظام دخول آمن:** تسجيل دخول وتصنيفات مستخدمين (أطباء ومرضى) مع تشفير كلمات المرور.
- **لوحة تحكم المريض:** - حجز مواعيد جديدة بسهولة.
  - متابعة حالة المواعيد (قيد الانتظار، مؤكدة، مكتملة).
  - عرض بيانات الحساب الشخصي.
- **لوحة تحكم الطبيب:** - عرض قائمة بجميع المواعيد المحجوزة.
  - إمكانية تحديث حالة الموعد مباشرة.
- **واجهة مستخدم متجاوبة:** يعتمد المشروع على Bootstrap 5 لضمان تجربة مستخدم سلسة على مختلف الأجهزة.

## التقنيات المستخدمة
- **Back-end:** PHP 8.x
- **Database:** MySQL
- **Front-end:** HTML5, CSS3, Bootstrap 5
- **Server:** Apache (XAMPP/WAMP)

## طريقة التشغيل
1. قم بوضع ملفات المشروع في مجلد `htdocs` داخل مجلد باسم `hospital`.
2. قم بتشغيل سيرفر Apache و MySQL من خلال برنامج XAMPP.
3. قم بإنشاء قاعدة بيانات جديدة باسم `hospital_db` عبر `phpMyAdmin`.
4. قم باستيراد ملف قاعدة البيانات (`database.sql`) الموجود ضمن المجلد.
5. افتح المتصفح وتوجه إلى الرابط: `http://localhost/hospital/login.php`.

## هيكلة قاعدة البيانات
- `users`: تخزين بيانات الأطباء والمرضى (الاسم، البريد، كلمة المرور المشفرة، الدور).
- `appointments`: تخزين تفاصيل المواعيد (المريض، الطبيب، التاريخ، الحالة).
